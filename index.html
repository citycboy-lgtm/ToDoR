                      How to Use ToDoR the Helper

                      1. Adding Tasks

                      Use the input fields at the bottom of each column (Daily, Weekly, etc.) to add new tasks. For daily tasks, you can now specify a duration in hours and minutes, and assign them to Morning, Day, or Evening.

                      2. Completing Tasks & Earning Stars

                      Click the checkbox next to a task to mark it as complete. This will earn you stars! The number of stars is determined by the star rating you set for that task (click the stars to change the rating, from 1 to 5).

                      3. Standard Tasks (Automation!)

                      Click "Manage Daily", "Manage Weekly", etc., to set up recurring tasks. These tasks will automatically appear on your daily list when they are due (e.g., every morning, every Monday, every 15th of the month).

                      4. The Rewards Shop

                      Click the "Rewards Shop" button to spend your hard-earned stars! You can add your own custom rewards, set their star cost, and redeem them whenever you have enough stars.

                      5. Daily Check-in & Star Cap

                      Each morning, the app will ask how your previous day was. Based on your answer and yesterday's star total, it will set a recommended star goal (Star Cap) for the day to help you stay motivated but not overwhelmed.

                      6. Add to Calendar

                      Click the calendar icon üìÖ next to a task to download a calendar file (.ics). You can import this file into Google Calendar, Apple Calendar, or any other calendar app to schedule your task.

                      7. Saving

                      Your progress is saved automatically to the cloud in real-time, so you can close the app and come back anytime!

                      Settings

                      App Permissions & Notifications

                      For reminders and alarms to work, especially on a mobile device, follow these steps:

                      1. Grant Permission: When your browser first asks for permission to show notifications, please click "Allow".
                      2. Add to Home Screen (Mobile): In your mobile browser menu, find the "Add to Home Screen" or "Install App" option. This makes the app work more reliably.
                      3. Check Battery Settings (Android): Go to your phone's Settings > Apps > [Your Browser] > Battery. Set it to "Unrestricted" to prevent the phone from stopping the app's reminders to save power.

                      Firebase Configuration

                      Update the Firebase project configuration used by this app. Warning: Changing this will connect you to a different database.

                      Export Data

                      Save a backup of all your tasks and rewards to a JSON file.

                      Import Data

                      Import tasks and rewards from a JSON file. Warning: This will overwrite all your current data.

                      Firebase Configuration Required

                      To run this app on your own, it needs a Firebase project to store data.

                      1. Create a new project in the Firebase Console.
                      2. Add a new Web App (</>).
                      3. Copy the `firebaseConfig` object and paste it below.
                      4. In your Firebase project, go to Firestore Database and create a database.

                      Your configuration is saved in this browser's local storage.
